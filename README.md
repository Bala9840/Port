## 🔧 Installation & Usage

Clone the repo:
```bash
git clone https://github.com/Bala9840/Port.git
````

Navigate to the project:

```bash
cd Port
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm start
```

Build for production:

```bash
npm run build
```
